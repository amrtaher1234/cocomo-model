Comoco Model with All of Its types

What is Commoco Models


The Constructive Cost Model (COCOMO) is a procedural software cost estimation model developed by Barry W. Boehm. The model parameters are derived from fitting a regression formula using data from historical projects (61 projects for COCOMO 81 and 163 projects for COCOMO II). COCOMO is used to estimate size, effort and duration based on the cost of the software.

Types of the models

Basic Model

Used for early rough estimates of project cost, productivity, performance and schedule

Intermediate Model

Takes basic cocomo as a starting point.
Uses effort adjustment factor which is calculated as the product of 15 multipliers (cost drivers).

Complete/Detailed Model

Uses different effort multipliers for each phase of a project.

Features

Interactive Model

Interactive cost estimation software package that models/predicts the cost, effort and schedule for a new software development activity.

Scaleable

Can be used on new systems or upgrades

Tested

Derived from statistical regression of data from a base of 63 past projects(2000 - 512,000 DSIs(delivered source instructions)).

Algorithmetic

Uses mathematical formula to predict project costs based on estimates of various types
